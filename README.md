# 🛡️ FUASK Sentinel

**Cybersecurity Monitoring & Threat Detection System** for Federal University of Applied Sciences, Kaduna (FUASK) website.

## Features
- Real-time website availability monitoring
- SSL Certificate expiration alerts
- Security headers analysis
- Anomaly & tampering detection
- Telegram notifications
- Daily security reports

## Commands
- `/start` - Welcome message
- `/status` - Run full security scan

## Tech Stack
- Python 3
- aiogram (Telegram)
- Requests + BeautifulSoup
- Cryptography & Security Analysis

## Setup
1. Copy `.env.example` to `.env` and fill in your bot token
2. `pip install -r requirements.txt`
3. `python main.py`

Made for academic project defense with strong cybersecurity focus.
